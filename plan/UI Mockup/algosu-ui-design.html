AI 개인 피드백
방문 노드를 set으로 관리하는 방식은 좋습니다. 다만 visited 초기화 시 start를 미리 추가했는데, 이 경우 다중 시작점 처리에서 의도치 않은 skip이 발생할 수 있습니다.
⟳
최적화 제안 코드
- visited = set([start])
+ visited = set()
- queue = deque([(start, [start])])
+ queue = deque([(start, [start])]); visited.add(start)
→ BFS 진입 전 명시적 마킹으로 의도 명확화
